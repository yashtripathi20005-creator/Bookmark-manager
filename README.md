# 📚 Bookmark Manager with Tags

A powerful CLI tool to manage your bookmarks with tag support, built with Python.

## ✨ Features

- Add, list, search, update, delete bookmarks
- Tag support (multiple tags per bookmark)
- Archive/unarchive bookmarks (soft delete)
- Import/export bookmarks to JSON
- Search by URL, title, description, or tags
- View statistics and all tags

## 🚀 Quick Install

```bash
git clone https://github.com/yourusername/bookmark-manager.git
cd bookmark-manager
pip install -r requirements.txt
